# Security Policy

## Unterstützte Versionen

| Version | Support |
|---------|---------|
| 1.x     | ✅ aktiv |

## Sicherheitslücken melden

Sicherheitslücken bitte **nicht** als öffentliches GitHub Issue melden.

Kontakt: rene@ruhrcoder.de

Bitte die folgenden Informationen mitliefern:
- Beschreibung der Schwachstelle
- Schritte zur Reproduktion
- Betroffene Version
- Mögliche Auswirkungen

## Sicherheitshinweis: Custom Code

Das Feld „Benutzerdefinierter Tracking-Code" im Admin führt eingegebenen JavaScript-Code **ungefiltert** aus. Zugriff auf diese Einstellung sollte auf vertrauenswürdige Administratoren beschränkt bleiben.
